# تصميم الوثائق البيداغوجية

مستودع مخصص لتطوير وتنظيم الوثائق البيداغوجية الخاصة بالتكوين المهني والتقني وفق المقاربة بالكفاءات (APC).

## أهداف المستودع

- تنظيم الموارد والوثائق البيداغوجية
- تطوير نماذج حديثة للدروس والتقييمات
- دعم التكوين المهني بالوثائق التطبيقية
- تحسين جودة إعداد الدروس والأنشطة
- توفير قوالب قابلة لإعادة الاستخدام

## محتوى المستودع

### مخططات الدروس
- مخطط الحصة التكوينية
- تخطيط المقاييس
- تنظيم الأنشطة التطبيقية

### وثائق التقييم
- شبكات التقييم
- معايير الأداء
- تقييم الأعمال التطبيقية
- نماذج الاختبارات

### موارد APC
- وثائق المقاربة بالكفاءات
- تنظيم الكفاءات
- مؤشرات الأداء
- الوضعيات التعليمية

### عروض ووسائل تعليمية
- عروض تقديمية
- وسائل شرح
- مخططات توضيحية
- موارد رقمية

### أدلة وتوجيهات
- دلائل استعمال
- توجيهات بيداغوجية
- تنظيم العملية التكوينية

## الفئة المستهدفة

- المكونون والمدربون
- أساتذة التكوين المهني
- المتربصون
- مهندسو التكوين
- المهتمون بالتعليم التقني والمهني

## أهداف التطوير المستقبلية

- دمج الذكاء الاصطناعي في إعداد الوثائق
- تطوير قوالب رقمية حديثة
- إنشاء مكتبة موارد بيداغوجية مفتوحة
- تحسين توثيق المشاريع التعليمية

## التقنيات والأدوات المستعملة

- برامج Microsoft Office 
- مجموعة Google Workspace
- برنامج Markdown
- برنامج Canva
- منصة Gamma App
- أدوات الذكاء الاصطناعي التعليمية
